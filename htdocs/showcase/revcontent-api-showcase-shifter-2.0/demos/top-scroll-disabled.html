RevShifter Revcontent Native API

Examples:

  • Default
  • Image Height - Set the height of the Shifter images.
  • On Load - Displays the shifter on page load.
  • Scroll Direction - reverse the scroll direction, showing when scrolling up.
  • Disable Scrolling - manually show Shifter or use the show_on_load option. Disabling scrolling forces the page to shift down to prevent overlaying content.
  • Top - Show the shifter on the top of the page.
  • Top Disable Scrolling - Show the shifter on the top of the page and disable scrolling. Show with show_on_load: true
  • Hide on Show Transition - Setting this to false will prevent the Shifter from hiding while it is showing
  • Touch Simulation - Simulate touch when the shifter shows to indicate touch support.
  • Visible Selector - Show when first element from provided selector is visible.
