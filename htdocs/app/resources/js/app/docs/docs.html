API Documentation

The RevContent API enables you to receive the same ad fill you would see in our traditional ad placements in a more flexible format, including json and xml.

URI / Endpoint

# Content-Type application/json or text/xml GET http://trends.revcontent.com/api/v1/

Request

All requests to the API are sent over HTTP and will utilize the following parameter set.

Property Type Default Description
api_key string Your account's assigned API key.
pub_id integer Your assigned account ID.
widget_id integer The ID of the widget associated with this request.
domain string The url encoded domain name associated with this widget.
user_ip string Request IP The end user's IP address required because response is tailored to end users geography.
referer string empty The end user's url encoded webpage url that the request is being made from.
width integer 1280 The end user's screen width.
img_w integer 420 The default creative image width.
img_h integer 315 The default creative image height.
format string json The format of the response, json or xml.
count integer 25 Number of content items you would like in the response.
offset integer 0 The content item that you would like the response to start from.
*Properties marked red are required.

Example

http curl
# HTTP GET Request http://trends.revcontent.com/api/v1/?api_key=53df2dd2d67ba93d0b262c86918fcbc19c2784d3&pub_id=45&widget_id=98&domain=bloomberg.com&user_ip=71.100.91.34&referer=www.revcontent.com&format=json&count=25&offset=0
# Sample cURL call with No Cache-control curl -X GET -H "Cache-Control: no-cache" http://trends.revcontent.com/api/v1/?api_key=53df2dd2d67ba93d0b262c86918fcbc19c2784d3&pub_id=45&widget_id=98&domain=bloomberg.com&user_ip=71.100.91.34&referer=www.revcontent.com&format=json&count=25&offset=0

Response

The expected response values in the return payload are shown below.

Property Description
headline Content headline.
url Content destination URL.
image URL of the content image.
brand Who is providing the content.
uid Unique content id to the system.
