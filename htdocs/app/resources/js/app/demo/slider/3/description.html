The per_row option allows you to define how many ads will be visible on a row. The total columns. This does not include the halved ad. per_row works on a breakpoint system to provide an adaptive UI

Available Options

per_row: { xxs: 1, xs: 1, sm: 3, md: 4, lg: 5, xl: 6, xxl: 7 }

Example

new RevSlider({ ... per_row: { xxs: 1, xs: 1, sm: 2, md: 2, lg: 3, xl: 4, xxl: 5 } });
