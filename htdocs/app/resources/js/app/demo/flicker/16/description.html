hide_provider is used to turn on/off the Creative Provider

Available Options

false (default)
true (Do NOT show Ad provider)

Example

new RevFlicker({ ... size: { margin: 2, image_height: false, headline_line_height: 18, headline_margin_top: 8, provider_line_height: 22, provider_margin_top: 8, provider_margin_bottom: 10, inner_margin: 10 } });
