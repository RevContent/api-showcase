Create RevExit with Provider Labels ON for "ALL" Ads

NOTE: This parameter depends on the I Parameter for Internal Ad Setting. For the purposes of this demo, i has been set to "rndm" for random placement of internal ad units.

NOTE: To trigger RevExit on desktop, simply leave mouseout of the page viewport, on Mobile/Tablet you must scroll first then wait approx 6 seconds.

  • dl Parameter = "Ads by My Network" (Demo #2)
  • po Parameter = "all"

Available options for this setting include:

  • all
  • sponsored
  • internal (Default)
  • off
